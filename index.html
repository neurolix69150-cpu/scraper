Installation et utilisation de WebCrawlr v2
🚀 Installation VPS (Docker)
curl -fsSL https://get.docker.com | sh sudo usermod -aG docker $USER && newgrp docker
scp -r scraper-dashboard user@VOTRE_IP:/home/user/ ssh user@VOTRE_IP cd scraper-dashboard
docker-compose up -d --build
http://VOTRE_IP:80
🔒 Sécuriser avec HTTPS + mot de passe
apt install caddy # Dans /etc/caddy/Caddyfile : scraper.votre-domaine.com { basicauth /* { admin $2a$14$votre_hash_bcrypt } reverse_proxy localhost:80 }
🎯 Sélecteurs CSS — Cheatsheet
# Par classe .product-title → tous les éléments avec class="product-title" .price, .cost → prix OU coût # Par attribut a[href] → tous les liens img[src] → toutes les images input[type="email"] → champs email # Combinés .card h2 → h2 à l'intérieur d'une .card table tr td:first-child → 1ère colonne d'un tableau ul.menu > li → li directs dans ul.menu # Nth .product:nth-child(2) → 2ème produit .item:last-child → dernier item
